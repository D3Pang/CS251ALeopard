Inventory

  • getItemListing():Item[]
    • returns a list of Inventory items
  • searchItem(String str):Item[]
    • search for items matching the search string in Inventory and returns a list of items
  • confirmAvailability (PurchaseOrder PO):Boolean
    • checks availability of a particular item and updates PO status to IN_STOCK or OUT_OF_STOCK
  • cancelPurchase(PurchaseOrder PO):void
    • cancel purchase and return item(s) back to warehouse
  • fulfillOrder(PurchaseOrder PO):void
    • Updates inventory (if required) and initiate fulfillment and delivery

Shipping

  • getShippingOptions(PurchaseOrder PO):ShippingOptions[]
    • returns a list of Shipping Options like:
      [{"type": "One-day shipping", "cost": "23.99", "est delivery date": "2016/12/10"},
      {"type": "Standard shipping", "cost": "5.99", "est delivery date": "2016/12/30"}]
  • cancelOrder(PurchaseOrder PO):void
    • cancel an order
  • updatePOStatus(double POId):OrderStatus
    • called by Shipping to update PO Status in Fulfillment
  • Methods exposed by Fulfillment for Shipping System

Accounting

  • generateInvoice(String custId, PurchaseOrder po):String
    • generates an invoice and returns the invoice id
  • getInvoiceStatus(String invId):Invoice
    • returns the status of the invoice.
  • updateInvoice(String invId, CreditCard cc):void
    • Once payment is successful, marks the invoice as 'Paid' and stores payment information
  • refundInvoice(String invId):Boolean
    • processes refunds. Returns 'true' if process successfully initiated
